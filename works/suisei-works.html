Suisei Works

動画編集・デザイン・イラスト・Web制作の実績をまとめるために制作したポートフォリオサイトです。

Overview

動画編集・デザイン・イラスト・Web制作などの制作実績をまとめるために制作したポートフォリオサイトです。 作品一覧だけでなく、各作品の詳細ページを用意し、制作意図や担当範囲が伝わる構成にしました。

青、夜空、星、水、透明感のある世界観をベースに、サイト全体の雰囲気が統一されるようデザインしています。 Home、About、Works、作品詳細ページを分けることで、閲覧者が制作内容や活動内容を順番に理解しやすい構成を目指しました。

Concept

「夜空に浮かぶ作品集」のような印象を目指し、静かで透明感のある世界観を意識して制作しました。 背景動画やガラス風のカードUI、淡い光の表現を取り入れることで、Suisei Worksらしい雰囲気が伝わるようにしています。

単に作品を並べるだけではなく、閲覧者が作品ごとの内容や制作背景を読み進められるよう、詳細ページの文章量や見せ方も調整しました。 デザイン性と情報の見やすさの両立を意識しています。

サイト構成

Homeではサイト全体の入口として世界観を伝え、Aboutでは活動内容やプロフィールを整理しています。 Worksでは制作ジャンルごとに作品を一覧で見られるようにし、各作品詳細ページでは制作意図・担当範囲・使用ツールなどを掲載しています。

作品数が増えても整理しやすいよう、ページごとに役割を分け、同じレイアウトを使いながらも作品内容に合わせて見せ方を調整できる構成にしました。

制作情報

  • 制作区分：自主制作
  • 制作期間：約2か月（作業時間：約300〜400時間）
  • 担当範囲：サイト構成設計 / デザイン調整 / HTML・CSSコーディング / JavaScript実装 / レスポンシブ調整 / 作品詳細ページ制作
  • 使用技術：HTML / CSS / JavaScript
  • 公開形式：GitHub Pages
  • 制作形式：ポートフォリオサイト

制作ポイント

サイト全体の世界観が崩れないように、背景・装飾・カードUI・アニメーションの雰囲気を統一しました。 青系の色味や光の表現を使い、夜空や水を連想させる透明感のあるデザインになるよう調整しています。

また、作品詳細ページでは、画像や動画だけでなく、制作意図や工夫した点も伝わるように文章構成を整理しました。 スマートフォンでも見やすいよう、レイアウトや余白、画像サイズの調整も行っています。

Homeは入口、Aboutは自己紹介、Worksは作品一覧、作品詳細ページは個別説明というように役割を分け、 閲覧者が迷わず必要な情報にたどり着けるよう意識しました。
