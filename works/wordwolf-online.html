Word Wolf Online

オンラインで遊べるワードウルフゲームの制作です。

Overview

オンラインで複数人が参加できるワードウルフゲームです。 ルーム作成・参加、待機画面、話し合い、投票、結果表示など、 ゲーム進行に必要な画面と動線を制作しました。

担当範囲

  • ルーム作成・参加機能
  • Firebase Realtime Database 連携
  • プレイヤー一覧の同期
  • ホスト判定
  • 画面遷移の実装
  • UI調整

使用技術

HTML / CSS / JavaScript / Firebase / GitHub Pages

制作ポイント

Firebaseを使ってルーム情報やプレイヤー情報を同期し、 オンライン上で同じ進行状況を共有できるようにしました。 また、各画面の役割が分かりやすくなるように、 待機・話し合い・投票・結果の流れを整理しています。
